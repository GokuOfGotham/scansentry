## ScanSentry 1.0.2

ScanSentry is a local, on-demand Windows utility for reviewing junk/cache files,
startup entries, browser policy hijacks, and executable file reputation.

It is not a real-time antivirus engine. Keep Windows Security / Microsoft
Defender enabled for real-time protection.

### What's Included

- Publisher-style app metadata now uses `GokuofGotham`.
- Installer file metadata now shows `CompanyName: GokuofGotham`.
- The release includes `SHA256SUMS.txt` for checksum verification.
- The release workflow runs dependency audit, smoke test, checksum generation,
  and GitHub artifact attestation.

### Verify Before Running

Download only from this official GitHub release.

Verify the installer hash in PowerShell:

```powershell
Get-FileHash ".\ScanSentry.Setup.1.0.2.exe" -Algorithm SHA256
```

Expected SHA-256:

```text
54433efadaeb001bd918cef98a7264bc13911304a8d70b3972c04d9f8c032353
```

The same value is attached as `SHA256SUMS.txt`.

### Windows Security Warning

ScanSentry is currently unsigned. Windows may show **Unknown publisher** or a
Microsoft Defender SmartScreen warning because the app is new and does not yet
have a code-signing certificate.

After verifying the checksum and confirming the file came from this official
GitHub release:

1. On the SmartScreen screen, choose **More info**.
2. Choose **Run anyway**.
3. If User Account Control appears, confirm the app name is `ScanSentry`.

Do not bypass Windows warnings for copies downloaded from mirrors, links in
messages, or anywhere other than this repository's official releases.

Code signing is planned for a future release.
