## ScanSentry 1.0.1

This release refreshes the app branding and adds security hardening in the
Electron shell, IPC layer, dependencies, and GitHub release workflow.

### Verification

The installer is currently unsigned, so Windows may show an "Unknown publisher"
or Microsoft Defender SmartScreen warning. Download only from this official
GitHub release and verify the SHA-256 checksum before running.

```powershell
Get-FileHash ".\ScanSentry.Setup.1.0.1.exe" -Algorithm SHA256
```

Expected SHA-256:

```text
f5c9d4f021796ed62c7b19d0ec5204951b9b1e57ec1d3171c911667747428e1a
```

The same value is attached as `SHA256SUMS.txt`.

### Security Notes

- Renderer sandboxing is enabled.
- New window creation and off-app navigation are blocked.
- File and registry actions are limited to findings from the current scan.
- Dependencies were updated to Electron 43.1.1 and electron-builder 26.15.3.
- `npm audit` reported zero known vulnerabilities at build time.

Code signing is planned for a future release.
